# 🌾 AI-Based Crop Yield Prediction & Optimization

## 🚀 Live Application  
👉 (https://ai-crop-yield-prediction-ixdcwdmcv6gcztzes82a3n.streamlit.app/) 

---

## 📌 Overview  
This project is an AI-powered web application that predicts crop yield based on agricultural parameters and provides optimization suggestions to improve productivity. The system uses Machine Learning techniques to analyze factors such as rainfall, fertilizer usage, pesticide usage, and cultivation area.

---

## 🎯 Features  
- 📊 Crop yield prediction  
- 🌦️ Analysis based on environmental and agricultural inputs  
- ⚙️ Optimization of conditions for maximum yield  
- 🌍 State and season-based input selection  
- 💡 Smart suggestions for improving crop productivity  
- 🖥️ Interactive web interface  

---

## 🛠️ Tech Stack  
- Python  
- Streamlit  
- Scikit-learn  
- Pandas  
- NumPy  

---

## 📂 Dataset  
The dataset includes the following features:  
- Crop  
- Crop Year  
- Season  
- State  
- Area (hectares)  
- Production (metric tons)  
- Annual Rainfall (mm)  
- Fertilizer (kg)  
- Pesticide (kg)  
- Yield (production per unit area)  

---

## ⚙️ How It Works  
1. User uploads a dataset (CSV file)  
2. Data is preprocessed and encoded  
3. A Machine Learning model is trained  
4. User inputs conditions (crop, state, rainfall, etc.)  
5. System predicts yield  
6. Optimization module suggests best conditions
